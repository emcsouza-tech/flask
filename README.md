# FLASK

Sistema simples gamificado com Flask + HTML em sala de aula.

## Funcionalidades

- Cadastro e login
- Lista de desafios
- Pontuação
- Ranking

## Executar

pip install -r requirements.txt  
python app.py  

Acesse: http://localhost:5000

## Melhorias Futuras
